# st10436169poepart1

## Project Overview
This project is Part 1 of a web development assignment focused on building the foundation of a multi-page website using HTML.

The website was created for **Matthew Snyman's Music School**, a music school established in 2024 that offers practical and theory music lessons for guitar and piano students in Athlone, Cape Town, and online.

This project demonstrates foundational web development concepts such as:
- Creating multiple linked webpages
- Using navigation menus
- Structuring content with headings, paragraphs, and lists
- Adding images and media assets
- Organising files and folders correctly

---

## Outcomes

In this phase of the project, the following objectives were completed:

### Plan the Project
The project was planned around the needs of a music school website. The target audience includes:
- Beginner music students
- Intermediate and advanced learners
- Parents looking for music lessons for children
- Individuals interested in online music lessons

The main goals of the website are to:
- Introduce Matthew Snyman's Music School
- Provide information about services offered
- Allow users to access contact information
- Answer common questions through an FAQ page

---

### Research and Content Gathering
Research was conducted to gather:
- Information about music education services
- Suitable website content and structure
- Relevant images related to music lessons and instruments

Assets collected include:
- Guitar and piano images
- Instructor image
- Music-related visual content

---

### Create the HTML Structure
The website was developed using HTML5 and includes:
- Header/title section
- Navigation menus
- Main content areas
- Images and descriptive text
- Footer-style navigation at the bottom of each page

The project contains multiple interconnected webpages to improve navigation and user experience.

---

### Organise Files and Folders
A structured folder system was created to keep the project organised and manageable.

---

## Website Pages

### Home Page (`index.html`)
The homepage introduces the music school and provides an overview of:
- Guitar lessons
- Piano lessons
- Music theory lessons

### About Page (`about.html`)
Provides information about:
- The school's history
- Mission and vision
- Teaching approach
- Instructor information

### Services Page (`services.html`)
Details the services offered:
- Guitar lessons
- Piano lessons
- Music theory lessons
- Online lessons
- In-person lessons

### Contact Us Page (`contactus.html`)
Contains:
- Operating hours
- Contact number
- Email address

### FAQ Page (`faqs's.html`)
Answers common questions about:
- Lesson registration
- Instruments taught
- Online/in-person lessons
- Age groups
- Lesson duration
